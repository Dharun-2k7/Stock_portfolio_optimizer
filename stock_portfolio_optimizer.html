01 — Overview

The Problem Statement

Given a set of stocks with different costs and expected returns, select an optimal subset within a fixed budget to maximize total return. This maps directly to the 0/1 Knapsack Problem — a classic NP-hard optimization problem.

Problem Formulation

Let stocks S = {s₁, s₂, …, sₙ} each have a cost cᵢ (price per share) and expected annual return rᵢ. Given budget W, find a subset T ⊆ S such that Σ cᵢ ≤ W and Σ rᵢ is maximized.

02 — Algorithms

Two Strategies Compared

Both algorithms solve the same problem but with different trade-offs between accuracy and speed.

Greedy Algorithm
Dynamic Programming

Greedy by Return Ratio

Sort all stocks by their return-to-cost ratio (r/c) in descending order. Greedily pick the best-ratio stock until the budget is exhausted. Fast but not always optimal.

Time ComplexityO(n log n)
Space ComplexityO(n)
OptimalityNot guaranteed
Best ForLarge n, fast approx

Step-by-step

  • 01Compute ratio = return / price for each stock
  • 02Sort stocks in descending order of ratio
  • 03Initialize: budget_remaining = W, portfolio = []
  • 04Iterate: if cost ≤ budget_remaining, add stock and subtract cost
  • 05Return portfolio and total expected return

0/1 Knapsack (DP)

Build a 2D table dp[i][w] where each cell stores the maximum return achievable using the first i stocks with budget w. Guarantees the globally optimal solution.

Time ComplexityO(n × W)
Space ComplexityO(n × W)
OptimalityAlways optimal
Best ForExact solution needed

Step-by-step

  • 01Create dp table of size (n+1) × (W+1), init to 0
  • 02For each stock i from 1 to n:
  • 03  For each budget w from 0 to W:
  • 04    dp[i][w] = max(dp[i-1][w], dp[i-1][w-cost[i]] + return[i])
  • 05Traceback from dp[n][W] to recover selected stocks

# Greedy Portfolio Optimizer — O(n log n) # Selects stocks by highest return/cost ratio def greedy_optimizer(stocks, budget): """ stocks: list of dicts {'name', 'cost', 'return'} budget: int — total capital available returns: selected portfolio, total_return """ # Step 1: Compute return-to-cost ratio for each stock ranked = sorted( stocks, key=lambda s: s['return'] / s['cost'], reverse=True ) portfolio = [] remaining = budget total_return = 0 # Step 2: Greedily pick stocks that fit in budget for stock in ranked: if stock['cost'] <= remaining: portfolio.append(stock) remaining -= stock['cost'] total_return += stock['return'] return portfolio, total_return, budget - remaining
# 0/1 Knapsack DP — O(n × W) # Guarantees globally optimal portfolio def dp_optimizer(stocks, budget): n = len(stocks) W = budget # Step 1: Build DP table (n+1) x (W+1) dp = [[0] * (W + 1) for _ in range(n + 1)] # Step 2: Fill the table for i in range(1, n + 1): cost = stocks[i - 1]['cost'] ret = stocks[i - 1]['return'] for w in range(W + 1): if cost > w: # Can't include this stock dp[i][w] = dp[i - 1][w] else: # Max of: skip stock, or include it dp[i][w] = max( dp[i - 1][w], dp[i - 1][w - cost] + ret ) # Step 3: Traceback to find selected stocks selected = [] w = W for i in range(n, 0, -1): if dp[i][w] != dp[i - 1][w]: selected.append(stocks[i - 1]) w -= stocks[i - 1]['cost'] total_return = dp[n][W] total_cost = sum(s['cost'] for s in selected) return selected, total_return, total_cost
